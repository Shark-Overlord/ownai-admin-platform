# 一、用户模块
1. **注册 / 登录**
    -  用户注册账号、登录验证 
    -  密码加密存储
2. **会员管理**
    -  会员等级管理（普通/plus / pro） 
    -  会员特权：积分加成、专属内容访问 
3. **积分管理**
    -  查询用户积分余额 
    -  积分流水记录（获取/消费） 
    -  积分规则：购买返积分

# 二、作品与 Prompt 模块
1. **作品管理（Artwork）**
    -  新增/编辑/删除作品 
    -  上传封面图、视频、Prompt 
    -  设置大类 + 标签 
    -  设置价格（ 积分可兑换） 
2. **Prompt 管理**
    -  Prompt JSON 存储（数据库字段） 
    -  前端 API 可动态获取 JSON 
    -  支持购买/积分兑换后访问 
3. **大类（Category）管理**
    -  新增/编辑/删除大类 
    -  前端获取大类列表 
4. **标签（Tag）管理**
    -  新增/编辑/删除标签 
    -  多对多绑定作品 
    -  支持前端筛选使用

# 三、订单模块
1. **订单管理**
    -  创建订单（现金或积分） 
    -  支付状态管理（待支付/已支付/已完成） 
    -  支付回调处理（支付宝/微信/Stripe） 
2. **购买逻辑**
    -  现金支付 → 成功后返积分 
    -  积分兑换 → 扣除积分并记录流水 
3. **订单查询**
    -  用户历史订单 
    -  管理员查看所有订

# 四、动态发布模块
1. **作品列表 API**
    -  支持按大类 + 标签筛选 
    -  支持分页、排序、搜索 
2. **作品详情 API**
    -  返回封面、视频、Prompt JSON、价格、标签、大类 
3. **后台上传/更新**
    -  上传作品 → 写数据库 + 对象存储 URL 
    -  更新作品 → 前端 API 自动生效，无需改前端代码

# 五、积分系统模块
1. **积分获取**
    -  购买返积分（金额 × 系数） 
    -  完成每日签到、任务奖励 
2. **积分使用**
    -  兑换作品 / Prompt 
    -  解锁会员内容 
3. **积分流水**
    -  记录类型：购买返还、兑换消耗 
    -  关联订单或任务 ID 

# 六、媒体管理模块
1. **封面图、视频、Prompt 文件上传**
    -  上传到腾讯云 COS 
    -  返回文件 URL 给数据库存储 
2. **下载与访问**
    -  前端调用 API 获取 URL 
    -  通过权限控制，购买或积分兑换后可访问

# 七、系统管理与安全
1. **权限控制**
    -  管理员权限：作品、分类、标签、订单管理 
    -  普通用户权限：浏览、购买、积分兑换 
2. **日志管理**
    -  上传/修改/删除操作日志 
    -  积分流水、订单操作记录 
3. **API 文档**
    -  Swagger/OpenAPI 自动生成接口文档 
    -  方便前端调用和调试
