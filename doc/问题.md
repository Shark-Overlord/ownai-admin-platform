场景	接口	说明
获取当前用户	GET /user/get/login	获取登录用户信息（头像、昵称、角色、会员等级等）
更新个人信息	POST /user/update/my	修改昵称、头像、简介等
退出登录	POST /user/logout	后端无状态，前端也要同时清除 localStorage 的 token
文件上传	POST /file/upload	上传头像、作品封面、视频等
创建订单	POST /order/create	购买作品
取消订单	POST /order/cancel	取消待支付订单
我的订单	POST /order/my/list/page	查看自己的购买记录
创建会员订单	POST /member/order/create	购买/续费会员
取消会员订单	POST /member/cancel	取消待支付的会员订单
我的会员订单	POST /member/order/my/list/page	查看会员购买记录
积分概览	GET /point/me	我的积分余额、签到状态等
每日签到	POST /point/check-in	签到领积分
积分记录	POST /point/record/list/page	积分变动明细
